How to Use
1

Open the Tool

Open this file in Google Chrome browser for best experience.

2

Upload BOQ File

Click on "BOQ File" upload button and select the COMBINE xlsx file provided by your supervisor.

3

Upload Barcode Report

Click on "Barcode File" upload button and select the Barcode Report downloaded from ERP.

4

Run Match

Click the "Run Match" button. The tool will automatically process and compare both files.

5

Review Summary

Check the unit-wise summary table. Results are shown as MATCH, SHORT, or EXTRA.

6

Export Result

Click "Export Excel" to download the result file — Summary, Barcode with Unit, and BOQ Merged.

⚠️ Important: Always use the latest BOQ file and freshly downloaded Barcode Report from ERP for accurate results. Do not modify either file before uploading.

ℹ️ Tool Logic & Info ▼
Admin View Only

🗂 BOQ File Rules

  • Colored (Green) tabs = Released
  • White tabs = Not Released
  • MAIN sheet नंतरच्या sheets process
  • MAIN नसेल → Colored sheets only
  • POUR / SUMMARY / MAIN → Skip
  • KN1(2) → KN1 merge
  • KN2(2) → KN2 merge

🔢 Unit Mapping (BOQ → Barcode)

  • UTTAM → JB (pkg qty)
  • KN6 → GROOVE (area)
  • DIS ALU1 → DISP_ALU_P1
  • DIS ALU2 → DISP_ALU_P2
  • बाकी सगळे → Sheet Name as-is

📊 Barcode File Rules

  • BALANCEUNIT column required
  • NR = Not Released → Excluded
  • CL = Cancelled → Excluded
  • JB → qty from Height field
  • GROOVE → Area based match
  • IPO: 3718/3719/3720/3721

🔧 Width Normalization

  • Blank width → 0
  • Brackets removed: (225+150) → 225+150
  • DP code → W:0 L:200
  • DPA code → W:150 L:300
  • Both blank → Code-only match

📤 Output — 3 Sheets

  • SUMMARY — Unit-wise BOQ vs Barcode
  • BARCODE_WITH_UNIT — Barcode + ASSIGNED_UNIT
  • BOQ_MERGED — All BOQ sheets combined
  • ✅ MATCH = 0 diff
  • ❌ SHORT = Barcode कमी
  • ⚠ EXTRA = Barcode जास्त
